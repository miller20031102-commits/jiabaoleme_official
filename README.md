## v6.7.19 好康入口優化

本版重點：
- 首頁新增 4 個快速入口：篩選、今日好康、投稿店家、支持。
- 新增「嘉義今日好康」測試版區塊，先以店家投稿 / 新品 / 公告 / 學生友善方向呈現，不放未確認的假優惠。
- 新增手機版底部快速導覽，方便使用者快速跳到篩選、好康與投稿區。
- 推薦店家入口改成「推薦 / 回報 / 投稿好康」，讓店家可以回報分類，也可以提供優惠、新品、期間限定。
- 修正統計文案，總店數改由 restaurants.js 自動計算，避免頁面數字跟資料庫不同步。
- 雞肉飯模式文案改用實際清單數量，不再寫死 25 間。

上線建議：
- 好康區目前是測試入口，不是實際優惠券清單；店家提供並確認後再新增真實優惠。
- 未來好康活動建議設定 7～14 天期限，避免過期資訊影響信任感。
