# 嘉飽了沒？正式版

嘉義市吃什麼骰子網站。

## 檔案
- index.html
- style.css
- script.js
- restaurants.js

## 功能
- 手機版優先
- 可愛風格
- 時段篩選：早午餐、午餐、晚餐、宵夜
- 分類篩選：咖哩、燒肉、火鍋、美式、日式、中式、小吃、咖啡甜點
- 預算篩選
- 隨便啦模式
- Google 地圖搜尋連結
- 深色模式

## 上線
把所有檔案放到 GitHub repository 根目錄，開啟 GitHub Pages 即可。


## 新增功能
- ❤️ 收藏功能：存在使用者自己的瀏覽器，不需要登入
- 🎲 從收藏裡骰一間：收藏多間店後可直接隨機抽
- 🔥 今日 TOP5：統計此裝置今天骰到最多次的店家，每天自動重新開始
- 以上功能都不需要資料庫，可直接部署到 GitHub Pages
